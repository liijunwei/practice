# 理解全文搜索

+ 第八章介绍的 `LIKE` 关键字可以用通配符匹配文本
+ 第九章介绍的 `REGEXP` 关键字可以用通配符匹配文本

+ 他们有几个重要限制
    + 性能低
    + 控制难
    + 结果不够智能



```sql
select note_text from productnotes where match(note_text) against('rabbit')
select note_text, match(note_text) against('rabbit') as rank from productnotes;

```
