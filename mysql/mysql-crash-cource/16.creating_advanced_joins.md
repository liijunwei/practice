# 使用表别名

+ 理由
    + 缩短SQL语句
    + 允许在单条select语句中多次使用相同的表名

```sql
SELECT
  cust_name,
  cust_contact
FROM
  customers AS c,
  orders AS o,
  orderitems AS oi
WHERE
  c.cust_id = o.cust_id
  AND oi.order_num = o.order_num
  AND prod_id = 'TNT2';
```
