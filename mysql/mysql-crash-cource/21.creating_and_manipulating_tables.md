# 创建表

+ 使用工具创建表
+ 使用命令行指令创建表

## 表创建基础

[参考](./00.create.sql)

示例

```sql
CREATE TABLE productnotes (
  note_id    INT         NOT NULL auto_increment,
  prod_id    CHAR ( 10 ) NOT NULL,
  note_date  datetime    NOT NULL,
  note_text  text            NULL,
  PRIMARY KEY (note_id),
FULLTEXT ( note_text )
) ENGINE = myISAM;
```

## 使用NULL值

+ 允许NULL值的列也允许在插入行时不给出该列的值

+ 不允许NULL值的列不接受该列没有值的行, 换句话说, 在插入或更新行时, 该列必须有值
